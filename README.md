# WebMuseum

Bienvenue dans le projet de musée développé avec Angular

# Installation

```bash
git clone https://github.com/challenge-studi/web-museum.git
```

```bash
npm install
```

## Development server

To start a local development server, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.

## Presentation des outils utilisés

    SonarLint : Pour assurer un code propre et de haute qualité.
    Gitmoji : Pour des messages de commit amusants et informatifs.
    Jira : Pour la gestion du projet.
    Confluence : pour le partage de documentation

## Déploiement

Le déploiement continu est configuré avec GitHub Actions.
